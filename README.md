# CYBV-498-Budget-IoT-Devices-and-Their-Vulnerabilities
Repository of final Cyber Operations Capstone. Included is the script created for the report, instructions for running the script, and the report itself.

Program tested on MacOS Tahoe 26.4.1

REQUIRED PYTHON PACKAGES:
1.python-nmap
2.Scapy

REQUIRED DEPENDENCIES FOR MACOS
1. Homebrew (https://github.com/homebrew/brew)
2. NMap (to install on MacOS use "brew install nmap")
3. Python 3.6+

Root privilege required to run network sniffing components.
Run via:
sudo python3 iot_scanner.py
